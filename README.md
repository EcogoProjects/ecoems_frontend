# Ecoems App frontend


## Instalación 

Instala las dependencias:

Las siguientes dependencias pertenencen a los iconos utilizados en la app:
```bash
npm install react-icons
npm install @boxicons/react
```
Las siguientes dependencias pertenencen a los gráficos:
```bash
npm install recharts
```
Las siguientes dependecias pertenecen a las análiticas de vercel:

```bash
npm i @vercel/analytics
```

Las siguientes dependencias pertenecen a la conexión con supabase:
```bash
npm install @supabase/supabase-js @supabase/ssr
```

## Ejecución
```bash
cd ecoems-app-frontend
npm run dev
```
